# Python ATM

A console-based ATM system built using Python.

## Features

- 🔐 PIN authentication with 3 login attempts
- 💰 Check account balance
- ➕ Deposit money
- ➖ Withdraw money
- 🔄 Transfer money
- 📜 Transaction history
- 🔑 Change PIN

## Concepts Used

- Python functions
- Loops (`while`)
- Conditional statements (`if`, `elif`, `else`)
- `match` / `case`
- Lists
- Input validation
- Returning values from functions

## Future Improvements

- Save balance, PIN, and history using file handling
- Add exception handling (`try`/`except`)
- Support multiple user accounts
- Generate mini statements with date and time
- Improve the user interface
